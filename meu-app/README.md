# Frontend (React) — Build & Deploy via Pipeline

Projeto frontend genérico desenvolvido para a faculdade, com o objetivo de praticar:
- Estrutura básica de um app React
- Build de produção
- Deploy automatizado usando pipeline (CI/CD)
